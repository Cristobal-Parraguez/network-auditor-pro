# 🛡️ Network Auditor Pro v1.4

Herramienta de auditoría de red automatizada desarrollada en Python. Este proyecto permite identificar dispositivos activos en una red local y generar reportes estructurados.

## ✨ Características
- **Detección Automática:** Identifica dinámicamente el rango IP sin configuración manual.
- **Reportes JSON:** Exporta los resultados para análisis profesional.
- **Privacidad:** Configurado para proteger datos sensibles mediante `.gitignore`.

## 🛠️ Requisitos
- Python 3.14
- [Nmap](https://nmap.org/) instalado en el sistema.

## 🚀 Uso
1. Instala dependencias: `pip install python-nmap`
2. Ejecuta: `python main.py`
